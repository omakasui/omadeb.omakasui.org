# Fonts

Omadeb uses _CascadiaMono Nerd Font_ as the default font for the terminal, while the system interface keeps _Ubuntuâ€™s default fonts_.

If you want to change the terminal font, open the _Omadeb Menu_ with `Super + Alt + Space`, then go to _Style > Font_ and select a different option.

You can install additional popular programming fonts from _Install > Style > Font_ in the same menu.
